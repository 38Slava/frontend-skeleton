
## Install

1. git clone git@github.com:django-stars/frontend-skeleton.git
2. TODO set origin
3. npm install -g bower coffee grunt
4. npm install
5. bower install

## Supports

* es6 (babelify)
* coffee
* jsx
* jade (-)
* sass
* gulp watching
* browserify
